# 📚 Recipe Book Web App

A simple Recipe Book web application built using HTML, CSS, and JavaScript.  
This project allows users to add, view, and delete recipes in a simple and interactive way using browser local storage.

---

## 🚀 Features

- Add new recipes with name, ingredients, and steps
- View all added recipes
- Delete unwanted recipes
- Data saved using LocalStorage
- Simple and responsive UI
- Fully frontend based project (no backend)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage

1. Download or clone the project
2. Open folder
3. Open `index.html` in browser
4. Start using the app
